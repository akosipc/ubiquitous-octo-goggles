## BYB 

All Bets are off! Check it out at [betyaballs.com](betyaballs.com)

#### Version

```
	Node.js 10.20.1
	Yarn 1.22.4
	Next.js
```

#### Installation

```
  $ git clone git@github.com:akosipc/bet-ya-balls.git
  $ cd bet-ya-balls
```

#### Setup

```
  $ yarn install
```

#### Starting the application

```
  $ yarn run dev
```

#### How to run the test suite

```
  $ yarn run test
```
